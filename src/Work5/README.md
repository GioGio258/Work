## 代码逻辑

### 1. 初始化与参数设置

- **窗口设置**：800x600 分辨率
- **交互参数**：
  - `light_pos_x`, `light_pos_y`, `light_pos_z`：光源位置
  - `max_bounces`：最大弹射次数（默认 3）
- **材质常量**：
  - `MAT_DIFFUSE = 0`：漫反射材质
  - `MAT_MIRROR = 1`：镜面反射材质

### 2. 辅助函数

- **normalize(v)**：向量归一化
- **reflect(I, N)**：计算光线的反射方向

### 3. 几何体相交测试

- **intersect_sphere(ro, rd, center, radius)**：测试光线与球体相交
- **intersect_plane(ro, rd, plane_y)**：测试光线与水平无限大平面相交
- **scene_intersect(ro, rd)**：遍历场景，寻找最近交点，返回距离、法线、颜色和材质ID

### 4. 渲染内核

- **render()**：
  - 为每个像素生成相机光线
  - 使用迭代循环追踪光线路径（最多 max_bounces 次）
  - 处理镜面反射和漫反射材质
  - 实现硬阴影检测
  - 累加最终颜色

### 5. 主循环与交互

- **main()**：
  - 创建 GUI 窗口
  - 初始化光源位置和弹射次数
  - 主循环：执行渲染、绘制结果、显示交互面板

## 实现功能

1. **三维场景构建**：
   - 无限大平面（y = -1.0）：黑白棋盘格纹理
   - 红色漫反射球（左侧）：中心 (-1.2, 0.0, 0.0)，半径 1.0
   - 银色镜面球（右侧）：中心 (1.2, 0.0, 0.0)，半径 1.0

2. **迭代式光线弹射**：
   - 使用 for 循环代替递归（GPU友好）
   - 追踪光线路径，最多弹射 5 次
   - 维护光线吞吐量（throughput），每次镜面反射衰减 0.8

3. **硬阴影检测**：
   - 向光源发射暗影射线
   - 判断交点是否在阴影中

4. **浮点精度处理**：
   - 将反射射线和暗影射线的起点沿法线方向偏移 1e-4
   - 防止射线自相交（Shadow Acne）

5. **UI 交互面板**：
   - Light X/Y/Z：动态调整光源位置
   - Max Bounces：调整最大弹射次数

## 注意事项

- **后端选择**：如果 GPU 初始化失败，可以将 `ti.init(arch=ti.gpu)` 改为 `ti.init(arch=ti.cpu)`
- **性能优化**：使用 Taichi 的并行计算能力，渲染速度会比 CPU 版本快很多

- **弹射次数**：设置为 1 时无反射，设置大于 1 时会出现镜中世界效果

## 实验原理

### 1. 光线追踪基本流程

1. **生成射线**：从相机出发，为每个像素生成一条射线
2. **求交测试**：计算射线与场景中几何体的交点
3. **材质着色**：根据材质类型计算交点颜色
4. **递归/迭代追踪**：对于镜面材质，生成反射射线继续追踪

### 2. 迭代式光线弹射

由于 GPU 不擅长处理递归，使用 for 循环实现迭代追踪：

```python
for bounce in range(max_bounces):
    t, N, obj_color, mat_id = scene_intersect(ro, rd)
    
    if mat_id == MAT_MIRROR:
        ro = p + N * 1e-4  # 法线偏移防止自相交
        rd = reflect(rd, N)
        throughput *= 0.8 * obj_color
        # 继续循环
    
    elif mat_id == MAT_DIFFUSE:
        # 计算光照和阴影
        final_color += throughput * direct_light
        break  # 漫反射终止追踪
```

### 3. 硬阴影实现

向光源发射暗影射线，检测是否被其他物体遮挡：

```python
shadow_ray_orig = p + N * 1e-4  # 法线偏移
shadow_t, _, _, _ = scene_intersect(shadow_ray_orig, L)
dist_to_light = (light_pos - p).norm()
in_shadow = 1.0 if shadow_t < dist_to_light else 0.0
```

### 4. 材质系统

- **漫反射材质**：光线被均匀散射，计算环境光和漫反射光
- **镜面反射材质**：光线按反射定律反射，继续追踪反射射线

### 5. 浮点精度问题（Shadow Acne）

由于浮点数精度限制，射线可能与自身相交，产生黑色噪点。解决方案是将射线起点沿法线方向偏移一个极小值：

```python
ro = p + N * 1e-4  # 偏移 1e-4
```

### 6. 场景布局

- **相机**：位置 (0, 1, 5)，视角微微向下
- **光源**：位置可通过 UI 调整，默认为 (2, 4, 3)
- **地板**：位于 y = -1.0，使用棋盘格纹理
- **红球**：左侧漫反射球
- **银球**：右侧镜面反射球
