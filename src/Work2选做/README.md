# 实验二选做：3D 立方体透视旋转

## 项目简介

本项目实现了一个基于 Taichi 库的 3D 立方体透视旋转效果。通过鼠标键盘控制，可以从不同角度观察立方体，体验真实的 3D 空间感。

## 项目架构

```
Work2选做/
├── __init__.py       # 模块初始化文件
├── main.py           # 主程序文件
└── README.md         # 项目说明文档
```

## 代码逻辑

### 1. 核心函数

- **get\_model\_matrix(angle\_x, angle\_y, angle\_z)**：
  - 功能：接收三个旋转角度（角度制），返回绕 X、Y、Z 轴旋转的模型变换矩阵
  - 实现：分别计算绕 X、Y、Z 轴的旋转矩阵，然后组合得到最终的变换矩阵
- **get\_view\_matrix(eye\_pos)**：
  - 功能：接收相机位置（三维向量），返回视图变换矩阵
  - 实现：将相机平移至世界坐标系的原点
- **get\_projection\_matrix(eye\_fov, aspect\_ratio, zNear, zFar)**：
  - 功能：接收视场角、屏幕长宽比、近截面距离和远截面距离，返回透视投影矩阵
  - 实现：先将透视平截头体挤压为正交长方体，再进行正交投影
- **compute\_transform(angle\_x, angle\_y, angle\_z)**：
  - 功能：在并行架构上计算顶点的坐标变换
  - 实现：计算 MVP 矩阵，对每个顶点进行变换，将结果存储在 screen\_coords 中

### 2. 渲染逻辑

- **main()**：
  - 初始化立方体顶点（中心在原点，边长为2，顶点坐标在\[-1,1]之间）
  - 创建 GUI 窗口
  - 处理键盘输入（W, S, Q, E, A, D 键控制旋转，Esc 键退出）
  - 计算变换并绘制立方体的12条边

## 实现功能

1. **3D 立方体渲染**：使用线框模式绘制一个三维立方体
2. **多轴旋转**：支持绕 X、Y、Z 三个轴的旋转
3. **透视投影**：实现了真实的透视效果，增强空间感
4. **交互控制**：通过键盘按键控制立方体的旋转
5. **实时渲染**：使用 Taichi 的并行计算能力，实现实时渲染

## 操作说明

- **W 键**：绕 X 轴顺时针旋转
- **S 键**：绕 X 轴逆时针旋转
- **Q 键**：绕 Y 轴顺时针旋转
- **E 键**：绕 Y 轴逆时针旋转
- **A 键**：绕 Z 轴顺时针旋转
- **D 键**：绕 Z 轴逆时针旋转
- **Esc 键**：退出程序

## 技术栈

- **Python 3.10+**：基础编程语言
- **Taichi**：GPU 并行计算库，用于加速渲染
- **Taichi GUI**：用于显示和交互

## 运行方法

```bash
uv run -m src.Work2.main
```

或直接运行 main.py 文件：

```bash
python src\Work2\main.py
```
